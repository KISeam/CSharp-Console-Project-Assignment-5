# C-Console-Project-Assignment-5

A simple C# console application developed using .NET 10.0.  
This project performs basic arithmetic operations using user input and conditional statements.

---

## 📌 Project Overview

The application takes two integer numbers and an arithmetic operator from the user, then performs the selected mathematical operation.

Supported operations:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)

The project demonstrates both:

- `switch` statement
- `if-else` statement

for handling calculator operations.
